# Wells_16Immune_57cells
Source code for "3D chromatin-based variant-to-gene maps across 57 human cell types reveal the cellular and genetic architecture of autoimmune disease susceptibility" (Trang et al, 2025)

Order of analyses steps:
1 - S-LDSC Enrichement analysis

2 - Variant-to-Gene (V2G) mapping

3 - TF motif analysis

4 - Enriched pathways analysis

5 - Colocalization analysis
